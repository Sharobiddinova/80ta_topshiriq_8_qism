# GitHub ga Yuklash

## 1. Git initsializatsiya
```bash
git init
git add .
git commit -m "Blockchain logistics system with 6 separate tasks"
```

## 2. GitHub repository yarating
- GitHub'da yangi repository oching (masalan: `logistics-blockchain-system`)

## 3. Remote ulash va push qilish
`YOUR_GITHUB_USERNAME` ni o'zingiznikiga almashtiring.

```bash
git branch -M main
git remote add origin https://github.com/YOUR_GITHUB_USERNAME/logistics-blockchain-system.git
git push -u origin main
```

## 4. Topshiriqni topshirish
- Ustozga shu repository linkini bering.
- Har bir topshiriq alohida faylda `tasks/` ichida ko'rsatilgan.
